Design Guidelines — Reference-Based & Professional B2B

Value Proposition Reference-Based + Professional B2B: drawing from Linear, Stripe, and GitHub — scientific authority combined with clear technical credibility. This document summarizes colors, typography, layout, components and accessibility for the Dr MO site.

Design Approach & Core Elements

Color, typography, layout and components

A. Color Palette

Dark Mode Primary (default)

  • Background: rgb(14, 23, 38)
  • Surface/Cards: rgb(20, 28, 47)
  • Primary Accent: var(--primary)
  • Secondary Accent: var(--secondary)
  • Text Primary: var(--text)
  • Text Muted: var(--muted)

B. Typography

  • H1 / Brand: 2.5rem, Bold
  • H2 / Sections: 2rem, Semibold
  • H3 / Subhead: 1.25rem, Medium (muted)
  • Body: 1rem, Regular (primary text)
  • Small: 0.85-0.9rem, muted
  • Font stack: system fonts for performance
